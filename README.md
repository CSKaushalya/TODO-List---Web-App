## üìù How to Use

1.  **Define Mission:** Enter your task in the input field.
2.  **Set Timeline:** Click the calendar icon to select a precise date/time.
3.  **Prioritize:** Assign a priority (High, Medium, Low) to color-code your badges.
4.  **Deploy:** Click "Add to Timeline" to save.
5.  **Analyze:** Use the Search bar or Export buttons to manage your task history.
